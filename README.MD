# Sample app

## Getting Started

### Prerequisites

Docker

### Installing

A step by step series of examples that tell you how to get a development env running

Build the images

```
docker-compose build
```

Run the app and server

```
docker-compose up app server
```

## Running the tests

To run the tests

```
docker-compose up app-test server-test
```
